# TEAM-PLEX v0 프로토타입

TEAM-PLEX는 게이머를 위한 깊이있는 커뮤니티를 제공해 사용자 스스로가 그 안에서 성장 할 수 있게하며, 도전 할 수 있는 계기를 제공한다.

## 계획
메이저 버전에 따른 팀플렉스 개발 방향:
* 0 : TEAM-PLEX 커뮤니티 프로토타이핑.
* 1 : TEAM-PLEX 커뮤니티.
* 2 : 래더/팀 관리 및 멀티미디어 컨텐츠.
* 3 : 리그 및 스폰서.
