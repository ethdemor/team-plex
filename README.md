# TEAM-PLEX v0 Prototype

TEAM-PLEX는 게이머를 위한 깊이있는 커뮤니티를 제공해 사용자 스스로가 그 안에서 성장 할 수 있게하며, 도전 할 수 있는 계기를 제공한다.

## TEAM-PLEX Major Version Plan
메이저 버전에 따른 팀플렉스 개발 방향:
* 0 : 프로토타입
* 1 : 커뮤니티
* 2 : 래더 및 방송
* 3 : TBD

## Current Objective
* 2015 : 프로토타입 완료.

## Versioning
`<major>.<minor>.<patch>`

버전관리 지침: [http://www.semver.org](http://www.semver.org)
