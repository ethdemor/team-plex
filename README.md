# TEAM-PLEX 0.0.1

TEAM-PLEX는 게이머를 위한 깊이있는 커뮤니티를 제공해 사용자 스스로가 그 안에서 성장 할 수 있게하며, 도전 할 수 있는 계기를 제공한다.

## TEAM-PLEX Major Version Plan
메이저 버전에 따른 팀플렉스 개발 방향:
* 0 : 프로토타입
* 1 : 커뮤니티/어드민
* 2 : 리그/래더/팀 관리
* 3 : 이적시장/글로벌

## Current Objective
* 2013 : 메이저 버전 0 개발 완료 및 [Heroku](http://www.heroku.com/) 배포
